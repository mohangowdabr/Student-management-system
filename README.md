# 🎓 Student Management System

## Overview

The Student Management System is a web-based application designed to efficiently manage student records, academic information, and administrative tasks. It provides a centralized platform for storing, updating, and retrieving student data while reducing manual paperwork and improving data accuracy.

## Features

* 👨‍🎓 Add new student records
* ✏️ Update student information
* 🗑️ Delete student records
* 🔍 Search students by ID or name
* 📋 View complete student database
* 📊 Manage academic details and performance
* 🔒 Secure data handling
* 📱 User-friendly interface

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

### 2. Navigate to the Project Directory

```bash
cd student-management-system
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Database

Create a database and import the SQL file:

```sql
CREATE DATABASE student_management;
```

### 5. Run the Application

```bash
python app.py
```

### 6. Open in Browser

```text
http://localhost:5000
```

## Usage

1. Launch the application.
2. Add student details such as name, roll number, department, and contact information.
3. View all registered students.
4. Update student information when required.
5. Delete records of inactive students.
6. Search and manage student data efficiently.
